2D Artist
My name is Fernanda Fantin, a Brazilian 2D / UI Artist currently living in Cambridge, UK. I have a bachelor's degree in Fashion Design but being so passionate about games made me start pursuing a career as a Game Artist.
Game Development
I'm currently co-developing "Cold Snap" with my partner using mainly Godot and Aseprite. Implementation and iteration are done after design and the required assets have been planned. The biggest challenges in this project are coding and achivable game design since our daily jobs are art focused. Thus, it is a fun and exciting challenge to overcome!
